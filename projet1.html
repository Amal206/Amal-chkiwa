Projet 1 - Gestion des Switchers

Explication du Projet

Ce projet consiste en la gestion des switchers via une interface web développée avec Django pour le backend et HTML, CSS, et JavaScript pour le frontend. L'application permet aux utilisateurs de configurer et de suivre l'état des différents appareils connectés dans un environnement dynamique.
