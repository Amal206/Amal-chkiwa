Gestion des Switchers

Interface web moderne pour la gestion et le monitoring des équipements réseau

À propos du Projet

Ce projet consiste en la gestion des switchers via une interface web développée avec Django pour le backend et HTML, CSS, et JavaScript pour le frontend. L'application permet aux utilisateurs de configurer, surveiller et suivre l'état des différents appareils connectés dans un environnement dynamique.

L'interface offre une vue d'ensemble en temps réel de tous les switchers connectés au réseau, avec des indicateurs visuels pour l'état de santé, la connectivité et les performances.

Les fonctionnalités incluent : surveillance en temps réel, configuration à distance, alertes automatiques, historique des événements et rapports détaillés.

Interfaces du Projet

Tableau de bord principal avec vue d'ensemble
Interface de configuration des ports réseau
Monitoring en temps réel du trafic
Gestion et configuration des VLAN
Rapports analytiques détaillés
